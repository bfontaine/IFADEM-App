Aide

1. Choix d’un pseudo

Les ressources que vous sélectionnez sont automatiquement associées à votre pseudo. Ceci permet de générer un flux de podcast unique quelque soit l’appareil que vous utilisez ou les ressources que vous sélectionnez.

2. Sélection des ressources

Touchez les noms de pays pour accéder à la liste des ressources correspondante. Vous pouvez afficher plus d’informations sur une ressource en touchant son image. Touchez une ressource pour l’ajouter ou la supprimer de votre sélection. Quand vous avez fini, validez en touchant le bouton en haut à droite. Attention : plus vous sélectionnez de ressources, plus le téléchargement sera important, donc assurez-vous de disposer d’une bonne connexion.

3. Flux de podcast

Une fois que vous avez validé votre sélection, l’adresse de votre flux de podcast est affichée. Copiez-la dans votre application de podcast, qui se chargera ensuite de télécharger automatiquement les ressources correspondantes. Il n’est nécessaire de copier cette adresse que la première fois (ou si vous changez de pseudo), l’application de podcast s’en souviendra pour les fois d’après.

4. Consultation des ressources

Assurez-vous d’être connecté à Internet quand vous sélectionnez de nouvelles ressources, puis ouvrez votre application de podcast pour les télécharger. Une fois les ressources téléchargées, vous pouvez les consulter en hors-ligne via votre application de podcast.
